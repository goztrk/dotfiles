# DOTFILES

This is a collection of custom settings for linux.
Right now sudo installations only works for ubuntu based distros.


## What's included?

- ZSH (as sudoer or normal user)
- OH-MY-ZSH
- Neovim
- Combination of NvChad and Christian Chiarulli's config
- tmux
- GIT aliases
- `bin` directory
